# 🤧 AI Allergy Detection System

An intelligent web application that detects **Food, Skin, and Environment-based 
allergies** using AI — supporting Voice, Chat, and Image inputs.

> Built as MCA Final Year Project | Fatima College, Madurai | 2024–2026

---

## 🚀 Features

- 🍔 Food Allergy Detection
- 🌿 Skin Allergy Detection
- 🌍 Environment Allergy Detection
- 🎤 Voice Input support
- 💬 AI Chatbot support
- 🖼️ Image Upload support
- 📊 Dashboard with risk score & trend graph
- 👤 User login, signup & profile
- 🌐 Tamil language support (Google Translate)
- ⚠️ Emergency alert for severe symptoms

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python, Flask |
| Database | MongoDB |
| AI / NLP | Custom chatbot engine, keyword-based NLP |
| Translation | Deep Translator (Google Translate) |
| Frontend | HTML, CSS, JavaScript |

## ⚙️ How to Run

**1. Clone the repo**
```bash
git clone https://github.com/Princy23-gif/ai-allergy-detection.git
cd ai-allergy-detection/backend
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Start MongoDB**
```bash
mongod
```

**4. Run the app**
```bash
python3 app.py
```

**5. Open browser**
